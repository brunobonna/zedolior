1. Quem usa o quê

👁️ Acompanhamento

Acessa a tela de monitor para ver em tempo real quantas pessoas estão em cada viagem, quem pagou e quem está reservado. Pode clicar no nome para ver o telefone e abrir o WhatsApp. Não altera nada.

⚙️ Operação (Admin)

Acessa o painel administrativo para aprovar pedidos, gerenciar passageiros, criar viagens e exportar listas. É quem efetivamente opera o sistema.

2. As três partes do sistema

🌐

Site público

Onde os clientes veem as viagens disponíveis e fazem pedidos de reserva. O cliente preenche os dados e o WhatsApp abre automaticamente com a mensagem pronta.

⚙️

Painel administrativo

Onde a operação acontece: aprovar pedidos, gerenciar passageiros, criar e encerrar viagens. Acesso por senha.

💡 Link: salve o endereço do painel admin nos favoritos do celular para acessar rápido.
👁️

Tela de monitor

Tela simples de acompanhamento com senha própria. Mostra as viagens ativas e o status dos passageiros. Nada pode ser alterado por aqui.

💡 Acesso: o endereço termina em /monitor.html — use a senha do monitor para entrar.

3. Fluxo do dia a dia

Toda reserva começa no site e segue este caminho:

🌐 Cliente acessa o site e preenche os dados dos passageiros
↓
📱 WhatsApp abre com a mensagem pronta — cliente envia para o motorista
↓
⚙️ Admin → aba Pendentes → revisar e aprovar o pedido
↓
⏳ Passageiro entra na lista como Reservado
↓
✅ Após pagamento: Admin → aba Passageiros → mudar status para Pago
📌 O sistema não processa pagamentos — a confirmação de pagamento é feita fora do sistema (dinheiro, PIX, etc.) e depois você atualiza o status manualmente.

4. Aprovar um pedido de reserva

Quando um cliente faz um pedido pelo site, ele aparece na aba 📬 Pendentes do painel admin.

Como fazer:

  1. No painel admin, clique em 📬 Pendentes no menu
  2. Veja a lista de pedidos aguardando aprovação
  3. Clique em ✅ Aprovar / Editar e confirmar no pedido
  4. Revise os dados do passageiro (nome, CPF, telefone, embarque, desembarque)
  5. Corrija qualquer dado errado se necessário
  6. Clique em ✅ Confirmar aprovação

O passageiro é adicionado automaticamente à viagem com status Reservado.

Se precisar recusar um pedido:

  1. Clique em ❌ Rejeitar
  2. Escreva o motivo (opcional)
  3. Confirme a rejeição
💡 Se não quiser aprovar agora, clique em ⏭️ Avaliar depois — o pedido continua na fila.
⚠️ Sempre confirme o pagamento por WhatsApp antes de mudar o status de Reservado para Pago.

5. Ver e gerenciar passageiros de uma viagem

Na aba 👥 Passageiros você vê todos os confirmados em cada viagem.

Para ver os passageiros:

  1. Clique em 👥 Passageiros no menu
  2. Escolha a viagem no seletor no topo da página
  3. Veja os contadores: total de vagas, disponíveis, pagos, reservados e pendentes
  4. A lista de passageiros aparece abaixo — clique em um nome para ver os detalhes

Marcar passageiro como Pago:

  1. Clique no nome do passageiro para expandir
  2. No campo Status da vaga, mude de ⏳ Reservado para ✅ Pago
  3. O sistema salva automaticamente — não precisa clicar em nenhum botão

Abrir WhatsApp do passageiro:

Clique em 💬 WhatsApp (número) dentro do card do passageiro.

Adicionar passageiro manualmente:

  1. Clique em ➕ Adicionar passageiro
  2. Preencha os dados e clique em Adicionar
📌 Passageiros adicionados manualmente entram com status Reservado por padrão. Lembre de mudar para Pago quando receber o pagamento.

6. Criar uma nova viagem

  1. Na aba 🗺️ Viagens, clique em ➕ Nova viagem
  2. Preencha a cidade de origem e a cidade de destino
  3. Se houver paradas intermediárias, clique em ➕ Parada e adicione as cidades na ordem do trajeto
  4. Defina a data e horário de saída
  5. Informe o total de vagas e o preço por pessoa
  6. Clique em ➕ Criar viagem

Observações

  • Observações internas: só aparecem no painel admin — use para lembretes da operação
  • Observações públicas: aparecem no site para os clientes — use para avisos importantes (ex: "ônibus executivo", "com parada para almoço")
💡 A viagem aparece imediatamente no site público assim que for criada com status Ativa.

7. Gerar lista para a empresa de ônibus

Antes de cada viagem, você pode gerar uma lista com todos os passageiros confirmados para enviar à empresa.

  1. Na aba 🗺️ Viagens, encontre a viagem desejada e clique nela para expandir
  2. Clique em 📋 Lista para empresa de ônibus
  3. Uma caixa de texto aparece com a lista formatada
  4. Toque na caixa, selecione tudo (Ctrl+A no computador ou segure no celular) e copie
  5. Cole no WhatsApp ou e-mail e envie para a empresa
📌 A lista inclui apenas passageiros com status Pago ou Reservado. Passageiros pendentes não entram.

8. Cancelar ou concluir uma viagem

Diferença entre cancelar e concluir

  • ❌ Cancelar viagem — a viagem não acontecerá. Ela sai do site público, mas os passageiros ficam registrados no sistema.
  • 🏁 Concluir viagem — a viagem já aconteceu. Ela é arquivada e sai do site público.

Como fazer:

  1. Na aba 🗺️ Viagens, clique na viagem para expandir
  2. Clique no botão desejado (Cancelar ou Concluir)
  3. Uma mensagem de confirmação aparece com o número de passageiros afetados
  4. Confirme clicando em ✅ Sim
⚠️ Atenção: ao cancelar uma viagem com passageiros pagos, o sistema não avisa os clientes automaticamente. Você precisará entrar em contato com cada um pelo WhatsApp.

9. Tela de acompanhamento (somente leitura)

A tela de monitor permite acompanhar as viagens em tempo real, sem precisar acessar o painel admin.

Como acessar:

  1. Abra o link da tela de monitor no navegador (o endereço termina em /monitor.html)
  2. Digite a senha do monitor e clique em Entrar

O que você pode ver:

  • Todas as viagens ativas com data e rota
  • Contadores: ✅ Pagos ⏳ Reservados 🔔 Pendentes
  • Ao clicar em um contador, a lista de nomes aparece
  • Ao clicar em um nome, aparecem os dados do passageiro
  • Se tiver telefone, o botão de WhatsApp abre a conversa direto
📌 A tela de monitor é somente para consulta. Para aprovar pedidos ou alterar dados, use o painel admin.
💡 Salve o link nos favoritos do celular — você pode adicionar na tela inicial como se fosse um aplicativo.
